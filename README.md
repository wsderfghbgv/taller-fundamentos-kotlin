Proyecto del taller con 30 ejercicios organizados por bloques tematicos.

- `src/ejercicios/Bloque1Variables.kt`

- `src/ejercicios/Bloque3Condicionales.kt`

## Descripcion breve

En este repositorio se implementan ejercicios de variables, null safety, condicionales, ciclos, funciones y colecciones en Kotlin, ejecutados desde un unico `main`.
